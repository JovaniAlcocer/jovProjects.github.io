Coffee Sales Project (Excel)

The Excel Project is located on Youtube, where the goal is to construct an interactive dashboard so that we can visually see the impact certain factors have on the sales of coffee.

With the given data, I first filled in the blank columns, such as the customer’s name, email and country, using the XLOOKUP function in Excel. While one may use the XLOOKUP formula to fill out the missing columns, I used a different function to fill columns I to L, by using the INDEX and MATCH functions. After filling out the missing columns with data from the other sheets, I then constructed new columns so that the viewer may better understand the terms being used to classify the different coffee types,size by using IF formulas. I then created a TABLE with the data I had worked on, and then I constructed PIVOT TABLES to make PIVOT CHARTS/GRAPH. With the data we were given, we formed graphs showing the most impactful stories, like different coffee types, sales over time(years) and more. Using SLICERS , we can visually see the impact certain qualities (for example country) has on the sales of coffee.
